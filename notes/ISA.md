# ISA
An ISA (Instruction Set Architecture) provides an abstract interface between hardware and low-level software


## Advantage:
allows different implementations to follow the same instruction set architecture (i.e. to run the same software) 

## Some ISAs:
- Intel x86, ARM, MIPS, PowerPC, SPARC
