# speed learning algorithm (sla)
1. max 1 min read/listen to: 
    - get a clue (intuition)
      - present
      - draw
      - link
    - extreme focus & relax
    
2. max 3 mins to:
    - develop clue
      - fundamental understanding
    - time limit reached
      - google
      - llm
      - friend
3. active recall:
    - immediately
    - write on paper (at least 1):
        - definition, 
        - motivation, 
        - meaning, 
        - linkages, 
        - application
        - ... 
4. attempt max(1, few) problems with [problem-solving algorithm](./speed_problem-solving_algo.md)
5. refine, recall, craftsmanship:
    - written to digital
6. attempt more problems with [problem-solving algorithm](./speed_problem-solving_algo.md)
