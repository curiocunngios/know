# Random variable
- ## Discrete
  - can be tricky but nothing very special. No need to memorize anything
- ## Continuous
  - just do integration for expected values $E(X)$ and turning into cdf and differentiation for converting from cdf

- ## Some facts
  - $E(b) =$ {{$b$}}
  - $E(aX + b) =$ {{$aE(X) + b$}}
  - $E(X + Y) =$ {{$E(X) + E(Y)$}}
  - $Var(b) =$ = {{$0$}}
  - $Var(aX) =$ = {{$a^2Var(X)$}}
  - $Var(aX + b) =$ = {{$a^2Var(X)$}}
  - if $X$ and $Y$ are independent, then $Var(X + Y) =$ = {{$Var(X) + Var(Y)$}}
