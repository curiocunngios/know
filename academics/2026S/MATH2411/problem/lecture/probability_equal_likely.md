# Problem 
- Suppose that a fair coin is thrown twice and the sequence of heads and tails is recorded.
- Let $E$ denote the event that at least one head is thrown. What is $P(E)$?

# Solution:
- The sample space is S={HH,HT,TH,TT}, each outcome with the probability 1/4.The event E={HH, HT, TH}, and its probability can be found by P(E)=3/4.

- **Remark:** The formula holds only if all the outcomes are equally likely. In this Example, if only the number of heads were recorded, then S would be {0, 1, 2}. These outcomes are NOT equally likely, and P(E) is not 2/3 .
