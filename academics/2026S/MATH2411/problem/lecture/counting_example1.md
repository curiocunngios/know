# Problem 
- Sam is going to assemble a computer by himself
- he has the choice of chips from 2 brands
- hard drive from 4
- memory from 3
- accessory bundle from 5 local stores

- how many different ways can Sam order the parts?

# Solution
$$
2 \times 4 \times 3 \times 5
$$
