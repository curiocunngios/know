# Problem
- There are two boxes. The first box contains 2 apples and 5 oranges. The second box contains 3 apples and 6 oranges. Now, we randomly select one fruit from the first box and put it into the second box. We then randomly select one fruit from the second box.
- Q: What is the probability that the two selected fruits are both apples?
