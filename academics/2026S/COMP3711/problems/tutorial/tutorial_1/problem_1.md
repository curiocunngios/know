# Problem 1
for each of the following statements, answer whether they are true or false:
- $1000n^2 + 1000n = O(n^3)$
  - true
- $n^2 - n = \Theta(n)$
  - false
- $n\log n = O(n^2)$
  - true
- $n\log n = \Theta(n^2)$
  - false
- $\frac{n}{100} = \Omega(n)$
  - true
- $12n + 2^n + n^3 = O(n^3)$
  - false
- $2n\log n + n = \Theta(n\log n)$
  - true
